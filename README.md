# Job-core-service

![Quarkus logo](https://quarkus.io/assets/images/quarkus_logo_horizontal_rgb_600px_reverse.png)

## Описание
    Сервис, отвечающий за управление сущностями с помощью rest api, и отправку сообщений об уведомлениях в очередь
    сообщений kafka.
